# Check if the Sentence Is Pangram

For more details, see the [LeetCode Problem Description](https://leetcode.com/problems/check-if-the-sentence-is-pangram/description/).

## Initial Approach

The initial approach leverages the properties of a set by converting the sentence into one. If the set reaches a length of 26, it means that all the characters of the alphabet are present in that sentence.
